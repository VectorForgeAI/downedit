Why down.edit?

Built for writers who value simplicity and privacy

Complete Privacy

Your documents never leave your computer. No cloud sync, no account, no analytics. Ever.

Lightning Fast

Built with Tauri and Rust. Under 15MB install, instant startup, minimal memory usage.

Export Anywhere

PDF with professional themes, Microsoft Word, or styled HTML. Share with anyone.

Visual Tables

Spreadsheet-style table editor. No syntax to memorize. Just click and type.

Never Lose Work

Auto-save and local version history. Browse and restore previous versions anytime.

Cross-Platform

Native apps for Windows, macOS, and Linux. Same experience everywhere.

Lightweight by design

Most Markdown editors are built with Electron, bundling an entire browser. down.edit uses Tauri with your system's native webview—resulting in an app that's 10x smaller and uses fraction of the memory.
